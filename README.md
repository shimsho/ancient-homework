
# Ancient Homework

Angular CRUD application using Apollo & GraphQl

### Main Packages ###
@angular/core  | ^15.2.0
------------- | -------------
typescript  | ~4.9.4
graphql | ^16
@apollo/client | ^3.0.0
@ngrx | ^15.3.0
rxjs | ~7.8.0

### Development ###

This repo uses yarn to manage dependencies.

To get the dev environment running on **localhost:4200**, run the following commands:

`yarn install`
`yarn start`
